Propriedade Transform 2D CSS

A propriedade Transform CSS aplica uma transformação 2D ou 3D a um elemento. Essa propriedade permite rotasionar, dimensionar, mover e distorcer elementos

Funções de transformação 2D do CSS

Com a propriedade CSS, você pode usar as seguntes funções de transformações 2D: transform

  • translate()
  • rotate()
  • scaleY()
  • scaleY()
  • scale()
  • skewX()
  • skewY()
  • skew()
  • matrix()

A propriedade translate() é usado para mover elemento de sua posição actula (de acordo com os paramétros que são passados no eixo X e eixo Y). Lembrando que podemos mover em eixos espeficos.
