Propriedade Scale

A propriedade scale() aumenta ou diminui o tamanho de um elemento (de acordo com os parametros fornecidos para largura e altura), lembrando que o valor inicial é (scale de 1).

Variações da Propriedae Scale

  • Scale(x,y)
  • ScaleX()
  • ScaleY()

Exemplo usando a propriedade scale para aumentar o tamanho do elemento

Scale(x,y)
ScaleX()
ScaleY()

Exemplo usando a propriedade scale para diminuir o tamanho do elemento

Scale(x,y)
ScaleX()
ScaleY()
